# RAG-Powered Customer Support Chatbot

A production-ready Retrieval-Augmented Generation (RAG) pipeline built with LangChain, OpenAI, and Pinecone for intelligent customer support automation.

## Tech Stack
- **LangChain** — orchestration and chain management
- **OpenAI API** — embeddings and GPT-4 completions
- **Pinecone** — vector database for semantic search
- **FastAPI** — REST API microservice
- **Docker** — containerized deployment
